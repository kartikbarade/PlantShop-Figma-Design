# PlantShop-Figma-Design
PlantShop is a modern and user-friendly UI/UX design created in Figma for an online plant shopping application. The project focuses on delivering a clean, minimal, and visually appealing shopping experience that makes browsing and purchasing plants simple and enjoyable.
The interface follows modern design principles with a nature-inspired color palette, intuitive navigation, and a responsive layout. Every screen is designed to provide a smooth user journey—from discovering plants to viewing product details and completing purchases.

✨ Features
Modern and minimal UI design
Clean and intuitive user experience
Nature-inspired color palette
Home screen with featured plants
Product listing and search interface
Product details page
Shopping cart interface
User-friendly navigation
Consistent typography and spacing
Reusable components and Auto Layout in Figma
